====================================================
Instrument Modules for Procedure and GUI Scripts
====================================================

The Instrument Libraries in this folder are usfull
for the different procedures and script in this directory.

As some of the modules were written presonally by me
and some were imported from Pymeasure these version are not perfect and do not contain all control possibilities. 

For example the MFLI does not work since the LabOne interface for a Windows 7 computer is just too old.

The dilution_connection module uses source code from Shilo's python interface
but not all. It was made so that connecting and communicating with the instrument is as easy as possible.

The keithley 2450 module appears several time since it is being used for different script and I never took the time to merge the versions.
